Frostbyte: Arctic Sunrise

Data Usage, Privacy Policy and continued legal stuff.

Data Usage

Frostbyte uses the following data:
- Your Discord ID
- All server IDs the bot is in
- Voice Channel regions (this is for language detection)

The NSFW filter will not be logged in Frostbyte. All images are sent off via URL format to deepai.org for processing.

Frostbyte will remove all information from the database when it is removed from a server. If you wish to prevent your data or a server from being added to the database, please contact me via email at hello@pandapa.ws, or via the support server, providing a server/user ID or name. However, please note this will cause heavy reduction in the bot's usage, including all server/user configuration and moderation features.

Privacy Policy

Frostbyte is aimed to be fully transparent with its data usage. All data is stored in a database, and is not shared with any third parties. The only exception to this is the NSFW filter, which uses deepai.org's API to process images. The bot and all services are hosted on a private server, and are not shared with any third parties. All services are hosted in the following locations:
- Frostbyte Stable: London, UK
- Frostbyte Alpha: Frankfurt, Germany
- Frostbyte Canary: Varies (note)
- Frostbyte Legacy: London, UK
- Frostbyte Website and Dashboard: London, UK
- Frostbyte Database: Dublin, Ireland (provided by MongoDB)

We use Oracle Cloud Infrastructure for all services, and all data is stored in the EU.
